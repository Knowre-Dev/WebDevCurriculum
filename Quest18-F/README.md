## Quest
* 텍스트 에디터 프로그램을 PWA 형태로 만들어 보세요.
  * 필요한 에셋을 적절히 캐싱하되, 버전업이 되었을 때 캐싱을 해제할 수 있는 형태가 되어야 합니다.
  * 해당 PWA를 데스크탑에 인스톨 가능하도록 만들어 보세요.
  * 오프라인인 경우에는 임시저장 기능을 만들어, 온라인인 경우를 감지하여 자동으로 서버에 반영되도록 만들어 보세요.
